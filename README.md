# Web 3 Deployer: Revolutionizing Decentralized Hosting and Deployment

**Web 3 Deployer** is an innovative platform designed to seamlessly integrate **decentralized hosting** with cutting-edge **blockchain technology**. By utilizing the power of decentralized storage systems like **IPFS** and smart contracts, this project empowers developers to host, deploy, and manage websites and decentralized applications (dApps) in a censorship-resistant, permanent, and decentralized manner.

## 🚀 Features & Key Highlights

- **Permanent Web Hosting**: Leverage blockchain technology to ensure your website is hosted permanently without the risk of censorship or downtime.
- **AI Integration**: Use artificial intelligence to enhance user experiences through automated decisions, personalized interactions, and intelligent processing.
- **Blockchain Security**: Secure your website with blockchain-based authentication systems and decentralized identity management.
- **Real-time Data & Analytics**: Track user interactions, visualize key metrics, and create real-time dashboards to engage your community.

## 🔨 Technologies Used

- **Next.js 14**: High-performance React framework for building modern web applications.
- **TailwindCSS**: Utility-first CSS framework to create responsive and highly customizable designs.
- **Drizzle ORM**: A simple yet powerful ORM designed to interact with decentralized data sources.
- **Google Gemini AI**: Cutting-edge AI integration to power intelligent features in your decentralized app.
- **Web3Auth**: A seamless solution for Web3 authentication, allowing users to sign in with their crypto wallet or other decentralized identity methods.
- **Neon Database**: A serverless database solution designed for the Web3 era, integrating easily with decentralized apps.
- **IPFS & Web3Storage**: Decentralized file storage solutions to host your assets on the distributed web, ensuring no central authority can censor or alter your content.

## 🏗️ How to Set Up Locally

1. **Clone the Repository**:
    Start by cloning this repo to your local machine.
    ```bash
    git clone https://github.com/Shyam-Prasath/Web-3-Deployer.git
    ```

2. **Install Dependencies**:
    Navigate into the project folder and install all required dependencies.
    ```bash
    cd Web-3-Deployer
    npm install
    ```

3. **Run the Development Server**:
    Start your development environment with the following command.
    ```bash
    npm run dev
    ```

4. **Access Your Application**:
    Open your browser and go to `http://localhost:3000` to see the app running locally.

## 🌐 Deploying to the Decentralized Web

To deploy your app in a decentralized way, follow these steps:

1. **Build the Project**:
    First, build the production version of your application.
    ```bash
    npm run build
    ```

2. **Upload to IPFS**:
    Use **IPFS** to upload your built assets and make your project available on the decentralized web. You can use tools like **Web3Storage** for this purpose, ensuring that your files are stored permanently and securely.
    - Visit [Gateway Pinata](https://api.gateway.pinata.com) and follow the steps to store your files.
